# 6-Pulse-Rectifier
MATLAB simulation and hardware sourcing for three optimized 6-pulse AC-DC rectifiers

https://quartz.b0b10.xyz/pe-project

## Design Specifications
The MATLAB simulation verifies continuous conduction mode (CCM) and calculates harmonic distortion to satisfy three distinct industry use cases:

* **Design 1: Cost-Optimized**
    * **Goal:** Total BOM cost < $45
    * **Performance:** > 98% efficiency, < 1% voltage ripple, > 0.89 power factor.
* **Design 2: High Performance**
    * **Goal:** Total BOM cost < $75
    * **Performance:** > 98.5% efficiency, < 0.25% voltage ripple, > 0.93 power factor.
* **Design 3: 400Hz Aircraft System**
    * **Goal:** Total BOM cost < $47 at a 400Hz input frequency.
    * **Performance:** > 98.8% efficiency, < 0.1% voltage ripple, > 0.945 power factor.
